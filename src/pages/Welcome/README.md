# Welcome page

## Functions:

---

`if !wallet_address`

  - open wallet connect modal

`if loading`

  - show spinner

`if !user`

  - upload avatar
  - set user name

`if user`

  - show Avatar
  - show user name

`else`

  - load investor data
  - load trader data
  - redirect to trader profile page
  - redirect to investor profile page
  - redirect to members list page
  - redirect to members cards page
  - open 'create fund' modal
